# 🗣️ PronounceAI

## AI-Powered English Pronunciation Learning App

### 📱 About
PronounceAI is a high-fidelity mobile application designed to help users improve their English pronunciation through AI-powered conversational avatars with real-time speech recognition.

### ✨ Features
- Real-time email validation (Error Prevention)
- Recognition-based UI with topic chips
- Progress tracking with visibility indicators
- AI avatar conversation with pronunciation feedback
- Hold-to-record microphone interface

### 🎨 Design Heuristics Implemented
- Error Prevention
- Recognition Rather Than Recall
- Visibility of System Status
- Aesthetic & Minimalist Design
- User Control and Freedom
- Help Users Recover from Errors

### 🛠️ Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome Icons
